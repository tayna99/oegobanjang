## Notion 페이지와의 차이

Notion 페이지는 2026-05-05 시점의 실행 정리라서 일부 표현이 당시 상태를 반영한다. 예를 들어 `hiring_agent.py`, `quota_tool.py`, Agent Runtime node가 아직 0바이트 또는 skeleton에 가깝다는 설명이 있었지만, 현재는 Mission 001 이후 작업으로 1차 구현과 테스트가 완료된 상태다.

| Notion 당시 표현 | 현재 상태 |
|---|---|
| Mission 001 acceptance가 아직 덜 닫힘 | `missions/completed/001-agent-runtime-skeleton.md`로 완료 |
| hiring_agent / quota_tool 실제 구현 필요 | 1차 구현 완료, executor 연결 완료 |
| eval fixture를 채워야 함 | intent, safety, workflow, rag, langchain judgment dataset이 채워짐 |

| real provider는 마지막 단계 | `missions/completed/013-real-llm-provider-integration.md`로 feature-flag/guard 수준 완료 |
