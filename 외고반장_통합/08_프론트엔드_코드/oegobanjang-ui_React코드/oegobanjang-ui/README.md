# 외고반장 프론트 UI 데모

스크린샷 기준으로 만든 React + Tailwind 프론트 프로토타입입니다.

## 실행

```bash
npm install
npm run dev
```

브라우저에서 Vite URL을 열면 됩니다.

## 포함 화면

- 오늘 할 일
- 채용 준비
- 근로자 목록
- 컨택 / 메시지 관리
- 리스크 케이스
- 행정사 검토 패키지
- 판단 기록

## 기술 스택

- React
- Vite
- Tailwind CSS
- lucide-react

현재는 화면 데모용 mock 데이터 기반입니다. API 연결은 `src/data.ts`를 API 응답으로 교체하면 됩니다.
