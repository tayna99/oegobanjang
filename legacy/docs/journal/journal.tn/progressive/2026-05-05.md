## 2026-05-05에 확정된 방향

- 현재 실행 기준은 예전 `phases/*`가 아니라 `missions/active/*.md`와 `missions/completed/*.md`다.
- 인력확보 에이전트는 후보 추천 AI가 아니라, 사업장 상태와 공식 근거를 바탕으로 고용 절차 초안, 체크리스트, 질문 목록, 위험 플래그를 만드는 agent다.
- AI가 자동으로 외부 발송, 정부 제출, 법률 판단, 국적 선호, 근로자 감시를 하지 않는다는 원칙을 코드와 테스트에 반영한다.
- RAG + LLM 판단 체인은 실제 provider 호출보다 먼저 Evidence Package, JSON schema, parser, fake client, deterministic report 순서로 닫는다.
