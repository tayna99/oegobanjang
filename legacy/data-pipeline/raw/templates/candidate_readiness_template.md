후보자 기본 확인 항목

- 여권 보유 여부
- 증명사진 제출 여부
- 건강검진 확인 여부
- 근무 가능 시점
- 희망 직무
- 언어/안내 필요 여부
- 기숙사 안내 여부
- 근무조건 안내 여부

여권 보유 여부 확인 항목

- 여권 보유 여부는 후보자 제출 준비도 확인 항목이다.
- 여권번호 원문은 RAG에 넣지 않고 DB/CSV에서 true/false로만 확인한다.

증명사진 제출 여부 확인 항목

- 증명사진 제출 여부는 서류 준비도 확인 항목이다.
- 사진 파일 원본이나 이미지 내용은 RAG에 넣지 않는다.

건강검진 확인 여부 확인 항목

- 건강검진은 확인/미확인 상태로만 표시한다.
- 건강정보 원문이나 상세 진단 내용은 RAG에 넣지 않는다.

근무 가능 시점 확인 항목

- 근무 가능일 입력 여부를 확인한다.
- 실제 날짜는 DB/CSV에서 읽고, RAG는 확인 필요 항목만 제공한다.

희망 직무 확인 항목

- 후보 희망 직무와 사업장 요청 직무의 일치 여부를 확인한다.
- 불일치 시 후보 평가가 아니라 추가 확인 필요로 표시한다.

언어/안내 필요 여부 확인 항목

- 후보자가 이해 가능한 언어와 안내 필요 여부를 확인한다.
- 언어권은 소통 준비를 위한 정보이며 선호/차별 추천에 사용하지 않는다.

기숙사 안내 여부 확인 항목

- 기숙사 제공 여부와 조건을 후보자에게 안내했는지 확인한다.
- 미안내 상태는 온보딩 안내 필요로 표시한다.

근무조건 안내 여부 확인 항목

- 주간, 야간, 주야 2교대 등 근무조건을 사전에 안내했는지 확인한다.
- 미안내 상태는 추가 확인 필요로 표시한다.

후보 제출 준비도 비교표 템플릿

후보 | 국적 | 희망 직무 | 근무 가능일 | 여권 | 사진 | 건강검진 | 숙소 안내 | 근무조건 안내 | 상태

금지 표현 정책

- 후보 B가 더 좋은 사람입니다.
- 후보 A가 더 성실해 보입니다.
- 베트남 후보가 더 낫습니다.
- 오래 일할 가능성이 높습니다.
- 이탈 가능성이 낮습니다.

허용 표현 정책

- 후보 B는 현재 제출 준비도가 더 높습니다.
- 후보 A는 사진과 건강검진 확인이 추가로 필요합니다.
- 후보 C는 근무 가능일 확인이 필요합니다.

후보 준비도 설명 원칙

- 후보를 추천하거나 순위를 매기지 않는다.
- 성실도, 이탈 가능성, 인성 판단을 하지 않는다.
- 제출 준비도와 추가 확인 필요 항목만 설명한다.
