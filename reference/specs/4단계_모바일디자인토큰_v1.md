# 4단계 — 모바일 디자인 토큰 시트 (v1)

> 작성일: 2026-07-04
> 기준: 사용자 제공 `DESIGN.md`(SOCAR 디자인 시스템)를 외고반장 모바일에 적용. 위험도·승인 상태색은 기능색(semantic)으로 유지.
> 소비처: 5단계 HTML 프로토타입이 이 CSS 변수를 그대로 사용한다.

---

## 1. 컬러

### 1.1 크롬 (무채색 — SOCAR 그대로)

```css
--color-canvas:        #ffffff;   /* 페이지·카드 배경 */
--color-surface:       #f2f3f8;   /* 입력 필, 보조 버튼 배경, 비활성 CTA */
--color-surface-dim:   #f5f5f5;   /* 화면 최하단 영역(탭바 위 구분 등) */
--color-text:          #354153;   /* 모든 본문·제목. 검정 금지 */
--color-text-muted:    #b4bbcb;   /* placeholder·비활성 라벨 전용. 활성 텍스트 사용 금지(대비 2.5:1) */
--color-hairline:      #e5e8ef;   /* 보더, 구분선, 스켈레톤 블록 */
```

### 1.2 브랜드

```css
--color-primary:       #0066FF;   /* 외고반장 블루. 화면당 primary CTA 1개 + 활성 탭 + 포커스 링만 */
--color-primary-press: #005EEB;
--color-on-primary:    #ffffff;
```

SOCAR의 "achromatic chrome, branded action" 원칙 적용: 블루가 넓은 면적(배너 배경, 아이콘 타일)에 깔리면 안 된다.

### 1.3 기능색 (semantic — 외고반장 고유, 배지·상태 전용)

```css
--sem-critical:  #EF4444;   /* CRITICAL, D+N/D-day, 반려 */
--sem-warning:   #F97316;   /* HIGH, D-1~30 */
--sem-pending:   #F59E0B;   /* 승인 대기, locked */
--sem-info:      #3B82F6;   /* MEDIUM, D-31~90, scheduled */
--sem-success:   #00BF40;   /* 승인 완료, done 스텝 */
--sem-neutral:   #6B7280;   /* LOW, 완료됨 */
```

사용 규칙: 배지 텍스트/보더/작은 도트에만. 배지 배경은 **각 색의 8% 틴트**(예: `rgba(239,68,68,.08)`)로 깔고 텍스트는 원색 — 넓은 원색 면 금지. 아이콘 타일·카드 배경·그라데이션 사용 금지.

## 2. 타이포그래피

폰트: `Pretendard Variable, Pretendard, -apple-system, sans-serif`. letter-spacing normal 고정(한글 트래킹 조정 금지). 색으로 위계를 나누지 않는다 — 크기·굵기로만.

| 토큰 | size/weight/lh | 용도 (외고반장 매핑) |
|---|---|---|
| `type-h1` | 26 / 700 / 1.38 | 화면 타이틀 1줄 ("오늘 브리핑", M5 "승인이 완료되었습니다") |
| `type-h2` | 22 / 700 / 1.36 | 섹션 타이틀, GreetingBanner 문장 |
| `type-h3` | 16 / 600 / 1.5 | 카드 제목, 바텀시트 블록 제목 |
| `type-body` | 16 / 400 / 1.5 | 본문, 초안 메시지, 요약 |
| `type-label` | 14 / 600 / 1.4 | 버튼 라벨, 탭 라벨, 필드 라벨 |
| `type-caption` | 13 / 400 / 1.4 | 배지 텍스트, 타임스탬프, 추천 이유, 안전 고지 |
| `type-num` | 15 / 700 / 1.0 | D-day·건수 숫자 (탭ular-nums) |

접근성 가산: 시스템 글꼴 크기 확대(Dynamic Type) 추종 필수. 200%까지 레이아웃 깨짐 없이 — 카드 높이는 고정값 금지, 내용 기반.

## 3. 스페이싱

```css
--sp-1: 4px;  --sp-2: 8px;  --sp-3: 12px;  --sp-4: 16px;
--sp-5: 20px; --sp-6: 24px; --sp-8: 32px;
```

| 자리 | 값 |
|---|---|
| 화면 좌우 가터 | 16 (좁은 기기 320px에서도 유지) |
| 카드 내부 패딩 | 20 (SOCAR card-region 준용) |
| 카드 사이 간격 | 12 |
| 섹션 사이 간격 | 24 |
| 버튼 패딩 | 12 × 18 |
| 바텀시트 내부 패딩 | 20, 상단 핸들 영역 12 |
| 탭바 높이 | 56 + safe-area |

## 4. Radius / Shadow

```css
--r-input: 12px;    /* 입력, 버튼, 커맨드 바 */
--r-chip: 14px;     /* 배지, 필터 칩, 통계 카드 */
--r-card: 16px;     /* 브리핑 카드, 결과 카드 */
--r-sheet: 20px;    /* 바텀시트 상단 모서리 (사다리 확장) */
--shadow-card: 0 4px 8px rgba(0,0,0,.10);   /* 유일한 그림자. 다층·유색 금지 */
--shadow-sheet: 0 -6px 24px rgba(0,0,0,.12); /* 바텀시트 전용 상향 그림자 */
```

필(9999px)·직각(0) 컴포넌트 금지. 카드 보더는 그림자 또는 `--color-hairline` 1px 중 하나만 (동시 사용 금지).

## 5. 터치 타깃 (40~60대 사용자 고려 — iOS 44px보다 상향)

| 요소 | 최소 크기 |
|---|---|
| 모든 탭 가능 요소 | **48×48px** (시각 크기가 작아도 히트 영역 확보) |
| primary CTA ([보내기 승인], [최종 승인]) | 높이 **52px**, 전폭 또는 ≥45% 폭 |
| 승인/반려 버튼 쌍 | 사이 간격 ≥ 12px (오탭 방지), 파괴적 액션([반려])은 항상 왼쪽·아웃라인 |
| 탭바 아이템 | 56px 높이 전체 히트 |
| 바텀시트 핸들 | 히트 영역 44px (시각 4×36 바) |
| 텍스트 링크(근거 보기 등) | 라인 히트 44px 확보 |

## 6. 컴포넌트 토큰 (핵심 6종)

```css
/* Primary CTA */
.btn-primary { background:var(--color-primary); color:#fff; border-radius:var(--r-input);
  padding:14px 18px; font:600 15px/1 Pretendard; min-height:52px; }
.btn-primary:disabled { background:var(--color-surface); color:var(--color-text-muted); }

/* Secondary (중립 필) */
.btn-secondary { background:var(--color-surface); color:var(--color-text);
  border-radius:var(--r-input); padding:14px 18px; font:600 15px/1; min-height:52px; }

/* 카드 */
.card { background:#fff; border:1px solid var(--color-hairline);
  border-radius:var(--r-card); padding:20px; }

/* 배지 */
.badge { border-radius:var(--r-chip); padding:3px 10px; font:600 13px/1.4;
  background:틴트8%; color:원색; }  /* 예: D-30 → --sem-warning */

/* 입력/커맨드 바 */
.input { background:var(--color-surface); border:none; border-radius:var(--r-input);
  padding:14px 16px; font:400 16px/1.5; color:var(--color-text); min-height:48px; }
.input::placeholder { color:var(--color-text-muted); }
.input:focus { outline:2px solid var(--color-primary); outline-offset:0; background:#fff; }

/* 바텀시트 */
.sheet { background:#fff; border-radius:var(--r-sheet) var(--r-sheet) 0 0;
  box-shadow:var(--shadow-sheet); padding:12px 20px calc(20px + safe-area); }
```

## 7. 모션

| 토큰 | 값 | 외고반장 용처 |
|---|---|---|
| `--motion-instant` | 0ms | 토글, 탭 전환 표시 |
| `--motion-fast` | 150ms | 버튼 press, 칩 선택, 배지 등장 |
| `--motion-standard` | 240ms | 바텀시트 rise, 카드 확장 — 기본값 |
| `--motion-slow` | 360ms | **M5 승인 완료 진입 전용** (push-in, 크로스페이드 금지 — "완료는 목적지") |
| `--ease-enter` | cubic-bezier(.2,0,0,1) | 들어오는 것 (시트, 결과 카드) |
| `--ease-exit` | cubic-bezier(.4,0,1,1) | 나가는 것 |
| `--ease-standard` | cubic-bezier(.4,0,.2,1) | 양방향 (아코디언, 탭) |

M9 스텝 스트리밍: 스텝 등장 `--motion-fast + --ease-enter`, 간격 300~600ms. `prefers-reduced-motion` 시 전 토큰 0ms + 스트리밍은 일괄 표시.

## 8. 상태 표현 (SOCAR §14 → 외고반장 매핑)

| 상태 | 처리 |
|---|---|
| 스켈레톤 | `--color-hairline` 블록, 카드 기하·radius 유지, 은은한 shimmer. **D-day·건수는 스켈레톤 금지 → `--` 표시** |
| 비활성 CTA | `--color-surface` 배경 + `--color-text-muted` 라벨. 기하 불변, 색만 변화 |
| 빈 상태 | 본문색 한 문장 + 보조 액션 1개. 일러스트·마스코트·이모지 금지 |
| 인라인 검증 오류 | 필드 아래 1줄, 무엇이 왜 + 무엇이면 되는지 ("체류만료일이 오늘보다 이전입니다") |
| 소액 성공 | 하단 3초 다크 토스트(`#354153` 배경, 흰 텍스트), 이모지 금지 |
| 승인 완료 | 전용 화면(M5). 토스트 축소 금지 |
| 오프라인 | 상단 배너 "인터넷 연결을 확인해 주세요 · 마지막 업데이트 HH:MM", 콘텐츠 유지, 승인 차단 |

## 9. 접근성 체크리스트

| 항목 | 기준 |
|---|---|
| 본문 대비 | #354153 on #fff = 9.4:1 (AAA) ✓ |
| `--color-text-muted` | 2.5:1 — placeholder/비활성 전용, 정보 전달 텍스트 금지 |
| 기능색 배지 | 틴트 배경 위 원색 텍스트 — 각 조합 AA(4.5:1) 검증 필수. 미달 시 텍스트를 한 단계 어둡게 (`#DC2626` 등) |
| 색맹 대응 | 위험도를 색으로만 구분 금지 — 배지에 라벨 텍스트(즉시 확인/우선 확인) 병기 (1단계 규칙 유지) |
| 포커스 | 모든 인터랙티브 요소에 `--color-primary` 2px 아웃라인 |
| 폰트 확대 | 200%까지 기능 유지 |
| 모션 | prefers-reduced-motion 전면 대응 |

## 10. 보이스 (문구 토큰 — SOCAR §10 준용 + 기존 가드레일)

- 선언형 단문. CTA는 명령형 동사: "승인하기", "등록", "다음" — "시작해보세요" 류 금지
- 트랜잭션 화면에 느낌표·이모지 금지
- 단정 금지(기존 원칙): "가능합니다" 대신 "검토가 필요합니다"
- 오류: 원인 1줄 + 행동 1개. "문제가 발생했습니다" 금지
- 안전 고지 고정 문구: "승인 전에는 외부 발송이 차단됩니다." (변경 불가)
